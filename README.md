Interactive Stopwatch Web App

A sleek, interactive stopwatch web application built with HTML, CSS, and JavaScript. Users can start, pause, record laps, reset, and export their lap times. The app also supports a theme toggle for light/dark mode.

Features

Start / Resume the stopwatch

Lap functionality to record split times

Reset the stopwatch to zero

Clear Laps to remove all recorded lap times

Export Laps to a CSV file for tracking

Theme Toggle between light and dark modes

Responsive and visually appealing interface

Demo

You can run the app locally by opening index.html in your browser.

Installation

Clone the repository:

git clone <repository-url>


Navigate into the project directory:

cd stopwatch-webapp


Open the index.html file in your browser:

open index.html


or simply double-click the file.

For live-reload experience, you can use Live Server in VS Code.

Usage

Click Start to begin timing.

Use Lap to record split times.

Pause with Resume and reset with Reset.

Manage lap records with Clear Laps or Export Laps.

Switch themes with the Theme button.
